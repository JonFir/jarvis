# Jarvis — Telegram‑бот, который превращает чат в терминал

Jarvis — это небольшой демон и CLI, которые принимают команды из Telegram и выполняют их на вашем Mac. Отвечает прямо в чат: stdout и stderr, без лишних танцев с SSH.

> ⚠️ Это инструмент с высоким уровнем доступа. Используйте только на своей машине/сервере и ограничивайте команды.

---

## Что умеет

- Принимает команды из Telegram и исполняет их на машине.
- Возвращает результат в чат (stdout/stderr).
- Работает как демон через launchd.

---

## Быстрый старт (macOS arm64)

### 1) Установка

```bash
curl -fsSL https://raw.githubusercontent.com/JonFir/jarvis/main/install.sh -o install.sh
bash install.sh
```

### 2) Запуск

После установки можно просто вызвать бинарь:

```bash
jarvisbot
```

Либо согласиться на установку демона — он будет стартовать автоматически при входе в систему.
